## 3. Wrap Your Crawler

```typescript
import { FetchBrain } from "@fetchbrain.com/sdk";
import { CheerioCrawler } from "crawlee";

const crawler = FetchBrain.enhance(
  new CheerioCrawler({
    requestHandler: async ({ $, request, pushData }) => {
      // This only runs when AI needs to "learn" (new page)
      const data = {
        title: $("h1").text(),
        price: $(".price").text(),
      };
      await pushData(data);
    },
  }),
  {
    apiKey: process.env.FETCHBRAIN_API_KEY,
    intelligence: "high",
    learning: true,
  }
);

await crawler.run(urls);
```

## 4. Run Your Scraper

```bash
FETCHBRAIN_API_KEY=fb_live_your_key npx ts-node scraper.ts
```
