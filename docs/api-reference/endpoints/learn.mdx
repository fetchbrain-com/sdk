## Endpoint

```http
POST https://api.fetchbrain.com/v1/learn
```

<Note>
  **Learning is always free.** You only pay when the AI successfully returns
  known data via the Query endpoint.
</Note>

## Request Body

```json
{
  "url": "https://example.com/product/123",
  "data": {
    "title": "Product Name",
    "price": "$99.99",
    "description": "Product description..."
  }
}
```

## Parameters

<ParamField body="url" type="string" required>
  The URL that was scraped
</ParamField>

<ParamField body="data" type="object" required>
  The scraped data to teach the AI
</ParamField>

<ParamField body="status" type="number" default="200">
  HTTP status code of the scraped page
</ParamField>

## Response

```json
{
  "success": true,
  "message": "Content learned successfully"
}
```

## Response Fields

| Field     | Type    | Description                |
| --------- | ------- | -------------------------- |
| `success` | boolean | Whether learning succeeded |
| `message` | string  | Status message             |

## Examples

<CodeGroup>
```bash cURL
curl -X POST "https://api.fetchbrain.com/v1/learn" \
  -H "Authorization: Bearer fb_live_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/product/123",
    "data": {
      "title": "Product Name",
      "price": "$99.99"
    }
  }'
```

```javascript JavaScript
const response = await fetch("https://api.fetchbrain.com/v1/learn", {
  method: "POST",
  headers: {
    Authorization: `Bearer ${process.env.FETCHBRAIN_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    url: "https://example.com/product/123",
    data: {
      title: "Product Name",
      price: "$99.99",
    },
  }),
});

const result = await response.json();
console.log("Learned:", result.success);
```

```python Python
import requests

response = requests.post(
    "https://api.fetchbrain.com/v1/learn",
    headers={
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json"
    },
    json={
        "url": "https://example.com/product/123",
        "data": {
            "title": "Product Name",
            "price": "$99.99"
        }
    }
)

result = response.json()
print(f"Success: {result['success']}")
```

</CodeGroup>

## Best Practices

1. **Batch learning** - Group multiple learns together when possible
2. **Clean data** - Remove unnecessary whitespace and formatting
3. **Include relevant fields** - Only teach data you'll query later
4. **Use consistent schemas** - Same fields for same page types
