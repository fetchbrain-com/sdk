## Endpoint

```http
GET https://api.fetchbrain.com/v1/stats
```

## Request

No query parameters required. Returns usage for the current billing period.

## Response

```json
{
  "queries": 46753,
  "recognized": 45230,
  "recognitionRate": 0.97,
  "learned": 1523,
  "period": "2026-06",
  "tier": "pro",
  "overageEnabled": true,
  "limits": {
    "quota": 500000,
    "hardCap": 750000,
    "quotaUsedPercent": 9.0,
    "hardCapUsedPercent": 6.0
  },
  "usageWarning": "none"
}
```

## Response Fields

| Field                       | Type   | Description                                              |
| --------------------------- | ------ | -------------------------------------------------------- |
| `queries`                   | number | Total queries this period                                |
| `recognized`                | number | Queries the AI knew (billable)                           |
| `recognitionRate`           | number | `recognized / queries`, clamped to 0–1                   |
| `learned`                   | number | New knowledge taught this period                         |
| `period`                    | string | Current billing period (`YYYY-MM`)                       |
| `tier`                      | string | Current plan tier                                        |
| `overageEnabled`            | boolean| Whether overage billing applies on this tier             |
| `limits.quota`              | number | Plan's included recognized queries                       |
| `limits.hardCap`            | number | Absolute monthly ceiling (blocks beyond this)            |
| `limits.quotaUsedPercent`   | number | Percent of quota used                                    |
| `limits.hardCapUsedPercent` | number | Percent of hard cap used                                 |
| `usageWarning`              | string | `none`, `warning` (75%+), `critical` (90%+), `exceeded`  |

## Examples

<CodeGroup>
```bash cURL
curl -X GET "https://api.fetchbrain.com/v1/stats" \
  -H "Authorization: Bearer fb_live_your_api_key"
```

```javascript JavaScript
const response = await fetch("https://api.fetchbrain.com/v1/stats", {
  headers: {
    Authorization: `Bearer ${process.env.FETCHBRAIN_API_KEY}`,
  },
});

const stats = await response.json();
console.log(`Recognition rate: ${(stats.recognitionRate * 100).toFixed(1)}%`);
console.log(`Quota used: ${stats.limits.quotaUsedPercent}%`);
```

```python Python
import requests

response = requests.get(
    "https://api.fetchbrain.com/v1/stats",
    headers={"Authorization": f"Bearer {API_KEY}"}
)

stats = response.json()
print(f"Recognition rate: {stats['recognitionRate'] * 100:.1f}%")
print(f"Quota used: {stats['limits']['quotaUsedPercent']}%")
```

</CodeGroup>

## Monitoring

Use the stats endpoint to:

- Monitor recognition rates
- Track usage against your quota and hard cap
- Set up alerts for overages
- Identify optimization opportunities

```javascript
// Example: Alert if recognition rate drops
const stats = await ai.stats();

if (stats.recognitionRate < 0.8) {
  console.warn("Low recognition rate - check if data changed");
}

if (stats.usageWarning === "critical") {
  console.warn("Approaching hard cap");
}
```
